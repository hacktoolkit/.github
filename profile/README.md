# Hacktoolkit

Hacktoolkit is a collection of reusable software foundations for building and shipping real products quickly.

We care about practical tools that are boring in the best way: clear conventions, repeatable project scaffolds, small utilities, and patterns that help teams move faster without turning every project into a one-off.

## What we are building

- **[django-htk](https://github.com/hacktoolkit/django-htk)** - reusable Django apps, utilities, middleware, and project patterns.
- **[expo-htk](https://github.com/hacktoolkit/expo-htk)** - utilities, services, models, and reusable design patterns for Expo and React Native apps.
- **[nextjs-htk](https://github.com/hacktoolkit/nextjs-htk)** - a TypeScript framework for building static Next.js sites with GitHub Pages deployment.
- **[flake8-htk-rules](https://github.com/hacktoolkit/flake8-htk-rules)** - Python lint rules for conventions used across Hacktoolkit projects.
- **[bash-ftw](https://github.com/hacktoolkit/bash-ftw)** - shell scripts, aliases, and bootstraps for productive local development.
- **[gimp-utils](https://github.com/hacktoolkit/gimp-utils)** - command-line image processing utilities built around GIMP headless workflows.

## Open source at Hacktoolkit

Hacktoolkit publishes shared foundations, experiments, reference implementations, and useful forks so others can learn from them, reuse them, and adapt them.

A few public places to start:

- [www.hacktoolkit.com](https://github.com/hacktoolkit/www.hacktoolkit.com) - the public Hacktoolkit site.
- [hexa.hacktoolkit.com](https://github.com/hacktoolkit/hexa.hacktoolkit.com) - Hexa, an AI coding companion built with Next.js, TypeScript, and Tailwind CSS.
- [openclaw-command-center](https://github.com/hacktoolkit/openclaw-command-center) - an AI assistant command-and-control dashboard.
- [chameleon](https://github.com/hacktoolkit/chameleon) - an adaptive web crawler for AI-assisted research workflows.
- [chrome-extensions](https://github.com/hacktoolkit/chrome-extensions) - a collection of Chrome extension projects and patterns.
- [code_challenges](https://github.com/hacktoolkit/code_challenges) - programming interview practice, take-home exercises, and challenge notes.

Some repositories here are active foundations used in current product work. Others are older experiments, educational projects, useful forks, or archives kept public because they may still help someone.

We also work in partnership with [Awe & Reverence](https://github.com/aweandreverence) product work, where Hacktoolkit patterns and tooling support Django, Expo, CSS, shell, and web foundations across Christian software projects.

## Our posture

We are builders who like sharp tools, readable code, and durable defaults.

That means the details matter: developer experience, accessibility, maintainability, clear licensing, responsible automation, and project structures that can survive past the first demo.

If you are exploring this organization, welcome. Use what helps, improve what can be improved, and build something worth maintaining.
